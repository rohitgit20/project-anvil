# Project Anvil

## Description
A robust task-based system built with Python and Docker. It uses CSV-based test cases to automate the validation of various programming tasks.

## Key Features
- **Task Automation**: 10 distinct programming tasks.
- **Dockerized**: Entire environment is containerized for consistent execution.
- **Testing**: Integrated automated testing using `pytest` and CSV data.

## Tech Stack
- **Language**: Python
- **Tools**: Docker, Git
- **Testing**: Pytest, CSV Parser

## How to Run
1. Clone the repo.
2. Run `docker build -t task1 .`
3. Execute `docker run task1`
